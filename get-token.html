Ожидание загрузки Clerk...
JWT токен:
Результат POST /api/chat:
⚠️ Clerk не загрузился (window.Clerk не определён).

Вероятная причина: браузер блокирует загрузку CDN-скриптов для страниц, открытых по протоколу file://.

Решение — открывайте через локальный http-сервер:

npx http-server . -p 8088 -c-1

Затем откройте: http://localhost:8088/get-token.html (с нужным ?__clerk_ticket=... в URL).
